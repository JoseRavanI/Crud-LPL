# CRUD RPG

## Tema Escolhido

Gerenciamento de personagens para RPG.

## O que o Sistema Faz

O sistema permite criar, listar, atualizar, deletar e comparar personagens de RPG. Cada personagem possui atributos de nome, força, defesa e vida. Os dados são salvos em um arquivo JSON e persistem entre execuções.

## Como o CRUD foi Aplicado

Create - Criar personagem: Adiciona novo personagem com validação de entrada
Read - Listar personagem: Exibe todos os personagens cadastrados
Update - Atualizar personagem: Permite modificar atributos específicos
Delete - Deletar personagem: Remove um personagem da lista

O sistema valida todas as entradas e pede novamente se digitar algo inválido.
